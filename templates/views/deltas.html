Delta & Snapshot Data

Deltas represent the changes between two snapshots. Each delta contains:

  • Added Records: New serials that weren't in the previous snapshot
  • Removed Records: Serials that were in the previous snapshot but not in the current one
  • Updated Records: Serials that exist in both snapshots but have changes in their data

Snapshots are point-in-time captures of all serials and their statuses. Each snapshot is created from an Excel file import.
